Visual Explanation in Deep Learning

Understanding the Network with Attention map

To interpret the deep learning in computer vision, visual explanation that visualizes the attention region in the inference process.
Visual explanation can be categorized into two approaches: gradient-based, which outputs an attention map using gradients, and response-based, which outputs an attention map using the response of the convolutional layer.

Modification Result of Attention map

To modify as many attention maps as possible, we created a tool that can modify attention maps.
This tool can add and remove an attention region simply by dragging the mouse.
These maps are modified so that an object or characteristic region with respect to a specific category is highlighted.

  • Details of the Experiment

    We use validation samples from the ImageNet dataset.
    Specifically, we replace the attention map during inference with the modified attention map and then check the changes of the classification results.
    As a network model, we use ResNet, which consists of 152 layers, along with the ABN (ResNet152+ABN).
    ResNet152+ABN is trained with 1,200k training samples from the ImageNet dataset.
    Then, we select samples from the validation samples and modify these attention maps.
